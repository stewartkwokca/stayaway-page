Instructions

There are five areas.

Each round is 1 hour, during which you can freely move between the five areas.

The players in the area with the least players move onto the next round.

All participants in the round in which all players are eliminated will be given the win.

Simple as that.

Discord Commands

  • `!area`: Check your current area.
  • `!areas`: Check number of players in each area.
  • `!end`: Check when the current round ends (time is in UTC).
  • `!help`: Returns list of commands (this text!)
  • `!id`: Check your id.
  • `!leaderboard`: Check global leaderboard
  • `!move n` (where `n` is the area number): Move between areas
  • `!score`: Check your score (number of wins)
  • `!start`: Create a profile (one Stay Away profile per discord account)

To start a DM with StayAway Bot, add a question mark before any command, like `?!start`.
